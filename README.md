# BHCG Leads Central

A Next.js app for BHCG to manage a centralized pitching database.

Features
- Login by user ID and password
- Dashboard with "Add to Database" form and your leads list
- LinkedIn ID-based autofill (mocked locally, no external AI)
- Central Database with filter/sort
- Leaderboard by leads added

Start by exploring `src/app/page.tsx` for the dashboard and `src/components` for UI.
